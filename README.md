# ⚡ Saulo Leão - Engenharia Elétrica

Uma Landing Page de alta conversão desenvolvida para um Engenheiro Eletricista sênior, focada em autoridade, segurança e geração de leads via WhatsApp. Este projeto foi construído com as tecnologias mais recentes do ecossistema React para garantir performance excepcional e uma experiência de usuário fluida.

---

## 🚀 Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no desenvolvimento web:

- **Next.js 15 (App Router):** Utilização de Server Components para performance e SEO otimizado.
- **React 19:** Aproveitando as últimas melhorias de renderização da biblioteca.
- **Tailwind CSS v4:** Estilização moderna com o novo motor de alto desempenho e variáveis de tema nativas.
- **Framer Motion:** Animações sutis e profissionais (fade-in, stagger, scroll reveal) que elevam a percepção de qualidade da interface.
- **Lucide React:** Conjunto de ícones leves e consistentes.
- **TypeScript:** Garantia de robustez e manutenção do código através de tipagem estática.

---

## 🎨 Destaques do Projeto

### 🎯 Foco em Conversão (UX/Copywriting)
- **CTAs Estratégicos:** Botões de conversão distribuídos em seções-chave (Hero, Problems, CTA Final).
- **WhatsApp Integration:** Links pré-configurados com mensagens personalizadas para rastreamento de origem.
- **Social Proof:** Seções dedicadas a depoimentos e métricas de autoridade (+15 anos de mercado).

### 📱 Design Responsivo & Moderno
- **Mobile-First:** Experiência otimizada para smartphones com menu overlay em tela cheia e navegação intuitiva.
- **Layout Profissional:** Paleta de cores baseada em Azul Escuro (confiança) e Âmbar (energia/eletricidade), com tipografia limpa (Geist Sans).
- **Sticky Navigation:** Menu de navegação que se adapta ao scroll para manter o acesso rápido às seções.

### ⚙️ Funcionalidades Técnicas
- **Scroll reveal animations:** Elementos que surgem suavemente conforme o usuário navega.
- **Floating Scroll-to-Top:** Botão intuitivo para retorno ao topo após leitura do conteúdo.
- **Optimized Performance:** Carregamento rápido de fontes e ícones, sem dependências desnecessárias.
